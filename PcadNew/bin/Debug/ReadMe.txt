1.По кнопке "Demo" активируется Демо режим с Демо данными.
2.Для начала работы с узлом после компоновки или последовательных методов выберите узел и нажмите кнопку "Загрузить".
3.В текстовом поле справа будут выводится шаги метода "Общий итерационный",текстовые поля выше предназначны
для вывода матриц R и D.
4.По кнопке "Start" весь алгоритм будет выполнен автоматически,от начала до конца.
5.По кнопке "Next" выполняется один шаг метода "Общий итерационный".
6.По кнопке "New" данные сбросятся в исходное положение(до выполнения метода "Общий итерационный").
7.По кнопке "Clear" все текстовые поля и ДРП будут очищены.
